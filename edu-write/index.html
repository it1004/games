💡 내보내기(다운로드) 후 실행 시 안내

미리보기 환경 외부(개인 PC 등)에서 실행할 때는 본인의 Google Gemini API 키가 필요합니다. 구글 계정만 있으면 무료로 발급받으실 수 있습니다.

🔑 무료 API 키 발급 방법:

  1. Google AI Studio 에 접속합니다.
  2. 로그인 후 'Get API key' 버튼을 클릭합니다.
  3. 생성된 키를 복사하여 위의 입력창에 붙여넣고 저장하세요.
